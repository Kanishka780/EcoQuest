# 🌍 EcoQuest

### Track • Understand • Reduce Your Carbon Footprint

EcoQuest is a modern sustainability platform built for **Google PromptWars Challenge 3** that helps individuals understand, track, simulate, and reduce their carbon footprint through personalized insights, habit tracking, gamification, and AI-powered sustainability guidance.

🚀 **Live Demo:** https://ecoquest-269406483122.asia-south2.run.app/login

---

## 📌 Problem Statement

Millions of people want to live sustainably but struggle to answer three important questions:

* How much carbon am I producing?
* Which activities contribute the most?
* What practical actions will make the biggest difference?

Most carbon calculators stop at awareness.

**EcoQuest goes further by helping users take action.**

The platform transforms sustainability into an engaging journey through footprint tracking, reduction planning, habit building, challenges, simulations, and personalized recommendations.

## 🚀 Local Setup

Clone the repository:

```bash
git clone https://github.com/Kanishka780/EcoQuest.git
cd EcoQuest
```

Install dependencies:

```bash
npm install
```

Create environment variables:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
GEMINI_API_KEY=
GOOGLE_MAPS_API_KEY=
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
